    • Creative Project

    Choose a node.js framework and npm package from the 1 million + ecosystem you would like to learn and build out a creative project that demonstrates the technologies usage and potential. Subject matter and technology is up to you, keep it academic, artistic, and creative and in the realm of web technology. Add git and make a github repo for your creative project, the README should serve as a comprehensive "how to" where someone else could learn from and use the technology and list the learning resources you used to build your project.
